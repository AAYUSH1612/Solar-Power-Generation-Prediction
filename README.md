# ☀️ SolarGen AI – Energy Intelligence Platform

SolarGen AI is a machine learning-powered web application designed to predict solar power generation and provide real-time insights into photovoltaic system performance. The platform integrates advanced ML models with an interactive Streamlit dashboard for intelligent energy monitoring and analysis.

---

## 🔍 Key Features

- ⚡ **Accurate Power Prediction** using trained ML models (XGBoost / ANN)
- 📊 **Interactive Dashboard** built with Streamlit
- 📈 **Real-Time Visualization** using Plotly graphs
- 🧠 **AI Insights** for peak generation prediction
- 🌡 **Thermal & Environmental Analysis**
- ⚙️ **Capacity Utilization Monitoring**
- 📅 **Time-Based Forecasting (Hourly Trends)**

---

## 🧠 Machine Learning Models Used

- Linear Regression
- Decision Tree Regressor
- XGBoost Regressor
- Artificial Neural Network (TensorFlow/Keras)

---

## 📊 Input Parameters

- Solar Irradiance (W/m²)
- Ambient Temperature (°C)
- Module Temperature (°C)
- Hour of the Day
- Day & Month

---

## 📈 Output Metrics

- Predicted AC Power Output (kW)
- Capacity Utilization (%)
- Operational Status (No Output / Suboptimal / Nominal / Peak)
- Peak Generation Hour Prediction

---

## 🛠 Tech Stack

- **Frontend:** Streamlit
- **Backend:** Python
- **ML Libraries:** Scikit-learn, XGBoost, TensorFlow
- **Visualization:** Plotly
- **Model Deployment:** Joblib

---

## 🚀 Deployment

The application is deployed using Streamlit Cloud for global accessibility.
